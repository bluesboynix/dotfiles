# Dotfiles

This repository contains my personal dotfiles for configuring my Linux environment, including:

- Neovim
- Foot terminal
- Hyprland
- Rofi
- Waybar
- Wlogout
- Yazi
- Zsh with Oh My Zsh
- Emacs
---

## Installation

To set up these dotfiles on a new machine:

1. Clone the repository:

git clone https://github.com/bluesboynix/dotfiles.git ~/dotfiles

## Requirements

- Neovim
  - ripgrep
  - fd
  - bat
  - tree
  - xdotool
  - xprop
  - wmctrl
  - gcc
  - clang
  - lazygit
  - delve
